# Claude 프로젝트 메모리 스냅샷

로컬 Claude Code 세션의 프로젝트 메모리(`~/.claude/projects/.../memory/`)를 저장소에 복사한 것입니다.
클라우드 세션(claude.ai/code 등)이나 다른 PC에서 작업을 이어갈 때 프로젝트 맥락을 유지하는 용도입니다.

- 인덱스: [MEMORY.md](MEMORY.md) — 메모리별 한 줄 요약
- 각 파일은 frontmatter(name/description/type) + 본문(사실, Why, How to apply) 구조
- 파일 안의 `[[이름]]`은 같은 폴더의 다른 메모리 파일을 가리키는 링크

주의: 메모리는 작성 시점의 관찰 기록입니다. 코드 동작·파일 경로·라인 번호는
현재 코드와 대조해 확인한 뒤 사용하세요. 로컬 메모리가 갱신되면 이 폴더도 다시 복사해 주세요.
(마지막 동기화: 2026-07-08)
